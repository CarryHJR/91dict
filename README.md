# install
```
conda create --name spider python=3.6
conda activate spider
pip install beautifulsoup4
pip install scrapy
pip install PyMySQL
```

# run
1. first modify password in `piplines.py`
2. enter `DictSpider` directory and run
```
scrapy crawl 91dict
```
